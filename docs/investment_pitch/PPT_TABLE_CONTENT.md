## 推荐使用的专业表达

### 替换"遇障就傻"的表述

**推荐表达**（按场景选择）：
1. **无法处理环境变化**
2. **无法适应非预期情况**
3. **需要人工干预异常处理**
4. **缺乏场景理解能力**
5. **规则依赖，适应性有限**

**完整句子示例**：
- "遇到环境变化需要重新编程"
- "非预设情况需要人工处理"
- "无法自主应对动态环境"

### 其他专业表达替换

| 避免使用 | 建议替换为 |
|---------|----------|
| 机器"傻"了 | 无法自主决策 |
| 停机"等死" | 等待人工干预 |
| "瞎"执行 | 规则执行 |
| "乱"来一通 | 无法处理非预期情况 |
